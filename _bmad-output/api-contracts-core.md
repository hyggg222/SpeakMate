# API Contracts - SpeakMate Core

## Overview
This document outlines the API endpoints discovered in the SpeakMate core application.

## Endpoints

### 1. Safe Mode Chat
- **Path:** `/api/safe-mode/chat`
- **Method:** `POST`
- **Description:** Handles real-time communication with the Gemini AI mentor. Supports multimodal input (text and audio).
- **Request Body:**
  - `messages`: Array of chat messages (role and content).
  - `data`: 
    - `audioBase64` (optional): Base64 encoded audio data.
    - `mimeType` (optional): The MIME type of the audio file (e.g., `audio/webm`).
- **Response:** `text/x-vercel-ai-data-stream` (Vercel AI SDK Data Stream).
- **AI Model:** `gemini-2.0-flash`
- **Features:**
  - Automatic silence detection in audio.
  - Persona enforcement (Ni - friendly English tutor).
  - Multilingual support (English primary, Vietnamese secondary).
  - Safety filtering (Harm categories blocked: NONE).
