# Project Overview - SpeakMate

## Mission
SpeakMate là một nền tảng EdTech tiên phong giúp học sinh Việt Nam xây dựng sự tự tin trong giao tiếp thông qua việc luyện tập với Mentor AI.

## Technology Stack
- **Framework:** Next.js 15+ (App Router)
- **Language:** TypeScript 5.x
- **Styling:** Tailwind CSS 4.0 + shadcn/ui
- **AI Engine:** Google Gemini 2.0 Flash (Multimodal)
- **Data & Storage:** Supabase (PostgreSQL + Storage)
- **ORM:** Drizzle ORM
- **Authentication:** Clerk
- **Streaming:** Vercel AI SDK

## Repository Type
**Monolith:** Một kho lưu trữ duy nhất chứa cả mã nguồn ứng dụng và các tài liệu kế hoạch chiến lược của BMAD.

## Key Features (MVP-0)
- **Push-to-Talk (PTT):** Ghi âm giọng nói và nhận phản hồi tức thì từ AI.
- **Safe Mode:** Môi trường luyện tập an toàn, không bị phán xét.
- **Living Transcript:** Hiển thị văn bản phản hồi theo thời gian thực (streaming).
- **Direct Base64 Upload:** Tối ưu hóa việc gửi dữ liệu âm thanh trực tiếp lên server AI.
