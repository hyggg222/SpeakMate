# Development Guide - SpeakMate

## Prerequisites
- Node.js v20+
- npm v10+
- Một tài khoản Supabase (cho Database & Storage)
- Một Google AI Studio API Key (cho Gemini)

## Setup Instructions
1. Di chuyển vào thư mục ứng dụng:
   ```bash
   cd speakmate
   ```
2. Cài đặt các gói phụ thuộc:
   ```bash
   npm install
   ```
3. Cấu hình biến môi trường:
   Sao chép `.env.example` thành `.env` và điền các thông tin sau:
   - `DATABASE_URL`: Đường dẫn PostgreSQL của bạn.
   - `GEMINI_API_KEY`: Key từ Google AI Studio.
   - `NEXT_PUBLIC_SUPABASE_URL`: URL dự án Supabase.
   - `SUPABASE_SERVICE_ROLE_KEY`: Key bí mật của Supabase (Service Role).

## Development Commands
- **Chạy server phát triển:** `npm run dev`
- **Xây dựng bản production:** `npm run build`
- **Chạy server production:** `npm start`
- **Kiểm tra kiểu dữ liệu:** `npm run typecheck`
- **Chạy tests:** `npx vitest run`

## Database Management
- **Tạo migration:** `npm run db:generate`
- **Đẩy schema lên DB:** `npm run db:push`
- **Giao diện quản lý DB (Studio):** `npm run db:studio`

## Project Architecture Patterns
- **Feature-based:** Mọi logic của tính năng Safe Mode nằm trong `src/features/safe-mode`.
- **Server Actions:** Dùng cho các tác vụ thay đổi dữ liệu (ví dụ: upload audio).
- **API Routes:** Dùng cho streaming dữ liệu AI.
